# Titanic Survival Prediction 🚢

Machine Learning classification project using the Titanic dataset from Kaggle.

## Project Objective

The objective of this project is to predict whether a passenger survived the Titanic disaster using different machine learning classification algorithms.

---

## Dataset

Dataset source:
- Kaggle Titanic - Machine Learning from Disaster

Main features:
- Pclass
- Sex
- Age
- Fare
- Parch
- IsAlone

---

## Machine Learning Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- XGBoost

---

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Feature Selection
5. Model Training
6. Hyperparameter Tuning
7. Overfitting / Underfitting Analysis
8. Model Comparison
